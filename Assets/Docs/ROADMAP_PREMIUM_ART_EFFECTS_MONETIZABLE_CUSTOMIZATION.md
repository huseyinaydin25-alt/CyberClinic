# Roadmap Addendum — Premium Art, Effects, and Monetizable Customization

**Date:** 2026-05-29  
**Status:** Added to roadmap  
**Scope:** Expand Cyber Clinic roadmap with premium artwork, effects, modular skins, customization, and cosmetic monetization foundations.

---

## Strategic reminder

Cyber Clinic is being planned as a premium, high-budget-equivalent mobile game.

The game must therefore include not only strong mechanics, but also high-value visual identity, satisfying effects, collectible customization, prestigious cosmetics, seasonal skins, and purchase-worthy cosmetic systems.

The artwork/effects layer is not decoration. It is part of retention, brand value, player identity, and monetization.


---

## Monetization design rule

Cosmetic systems should be monetizable, prestigious, and collectible, but should not break gameplay fairness.

Preferred approach:

```text
Cosmetics sell identity, prestige, expression, rarity, and seasonal exclusivity.
Core competitive outcomes should remain skill / strategy / progression based.
```

---

## Implementation rule

All cosmetic and shop systems must be built mock/offline-first before real payment SDKs are integrated.

This keeps the game testable, deterministic, and safe while preparing for later IAP, rewarded ads, subscriptions, season pass, and platform services.
